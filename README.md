# GitHub-User-Search App

## Project Description
This is a React-based web application that allows users to search for GitHub profiles by entering a username.  
The app fetches data from the GitHub API and displays user information dynamically.

This project was built as part of the ALX Frontend Engineering Capstone.

---

## Features
- Search GitHub users by username
- Fetch real-time data from GitHub API
- Display profile information
- Responsive user interface
- Error handling for invalid usernames

---

## Tech Stack
- React.js
- JavaScript (ES6+)
- CSS
- GitHub REST API
